遊び方

  • 「1人で」を選ぶと、タイムアタックでカードを揃えます。手数と経過時間が少ないほど好成績です。
  • 「2人で対戦」を選ぶと、同じ端末を2人で交互に操作します。ペアが揃うと続けて自分の手番、揃わないと相手の手番に移ります。
  • 「CPUと対戦」を選ぶと、あなたとCPUで対戦します。CPUは見たカードを一定確率で記憶しています。
  • 「オンライン対戦」を選ぶと、友達と2台の端末でリアルタイムに対戦できます。「部屋を作る」で表示される6桁のコードを友達に伝え、友達は「参加する」からそのコードを入力すると接続されます。
  • 全8ペア(16枚)を揃えるとクリア(対戦モードでは得点の多い方が勝ち)です。
  • 「最初から」ボタンでいつでも新しいカード配置に挑戦できます。
